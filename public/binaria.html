2: Padre Informático

Carlitos por suerte o por desgracia, le ha tocado tener un padre informático. Su padre, en el empeño de trasladar sus conocimientos a Carlitos decide enseñarle la conversión de un número decimal a uno binário.

Tras varios días practicando, el padre nota que Carlitos no está muy motivado, así que decide proponerle un juego a su hijo. Carlitos se vuelve loco por los caramelos, así que su padre decide usar esto a su favor. El juego es muy sencillo el padre le dirá al hijo un número y este tendrá que dividirlo en dos sumandos. Pero estos dos sumandos tienen que ser tales, que la suma de los 1 unos de los que constan al pasarlos a números binários sea máxima. Si el hijo gana se llevará un caramelo.

Tu labor aqui será hacerle un pequeño programa a Carlitos que le facilite la tarea (solo le serviremos de comprobación, pero no le diremos el resultado). El programa recibirá como primer parametro el número N que nos indica cuantos números le ha dado el padre a carlitos, despues recibirá los N números y devolverá para cada número entrante la mayor suma de unos posibles cuando se descompone en dos sumandos.

Breve aclaratorio: El padre le diría por ejemplo a Carlitos el número 6. Y Carlitos tendría que decirle que 6 = 3+3 ya que 3 en binário es 11 y 4 unos es la mayor cantidad de unos posibles para ese número dado. Nuestro programa debería devolver 4.

Ejemplo de input 

3 
1 
6 
2135

El 3 indica que vamos a recibir 3 números, que son: 1, 6 y el 2135

Guia del campeonato

Probar tu código:

Recuerda que debes tener instalado Nodejs. Una vez instalado haz lo siguiente:
  1. Abre la consola/terminal. En windows puedes darle a Inicio, teclear cmd y pulsar Enter.
  2. Ahora copia/pega este comando: 
    npm install seminarjs-contestclient -g

Para escribir tu solución al enunciado haz una carpeta en tu ordenador, y guarda dentro un archivo llamado por ejemplo ej1.js. Ahora vuelve a abrir la consola/terminal (instrucciones más arriba) y vete al directorio donde tienes el archivo, si estuviese en Documentos/Ejercicios harías cd Documentos/Ejercicios.

Finalmente ejecuta node ej1.js para ejecutar tu código en la consola. Para ver cómo leer el input del programa baja al paso "Programar la solución"

Enviar la solución:

Cuando tengas la solución lista y quieras probarla definitivamente, ejecuta el siguiente comando en la misma consola y directorio. Donde pone token tienes que poner tu correo electrónico hasta la @ sin incluirla.

seminarjsContest ejercicios.nichejs.com [token] [archivo.js]

No pongas los [], son para indicarte donde tienes que rellenar.

Programar la solución:

Para leer el input puedes usar el código a continuación, en el bucle donde esta el comentario dispones de la variable line que contiene el valor de esa linea.

var readline = require('readline');
var rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout
});

rl.on('line', function (line) {
  // Codigo aqui
});

Para mostrar las respuestas para cada linea usa console.log();

Si quieres programarlo en una función puedes usar lo siguiente:

var readline = require('readline');
var rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout
});

rl.on('line', function (line) {
  solucion(line);
});


function solucion(line){
  // Aqui tu solucion, que usa una linea de codigo
